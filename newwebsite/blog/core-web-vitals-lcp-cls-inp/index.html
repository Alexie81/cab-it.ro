Core Web Vitals: LCP, CLS și INP, explicate simplu

Cum influențează LCP, CLS și INP experiența vizitatorilor și ce merită optimizat întâi.

Publicat la 18.07.2026

Ce măsoară fiecare indicator

LCP descrie afișarea elementului principal, CLS stabilitatea vizuală, iar INP răspunsul la interacțiuni.

Praguri și optimizări

Țintele uzuale sunt LCP sub 2,5 secunde, CLS sub 0,1 și INP sub 200 ms. Comprimă imaginile, setează dimensiuni explicite, redu CSS-ul critic și elimină JavaScript-ul nefolosit.

Verifică datele reale

Testele de laborator ajută diagnosticul, dar hostingul, conexiunea și dispozitivele vizitatorilor influențează experiența reală.
